# DeutschBuch – Almanca Kitap E-Ticaret Sistemi

## Proje Hakkında

DeutschBuch, Almanca öğrenen öğrenciler için geliştirilmiş bir e-ticaret ve eğitim materyali yönetim sistemidir.

Bu proje İstanbul Topkapı Üniversitesi Yapay Zeka Yüksek Lisans Programı kapsamında İleri Web Programlama dersi final projesi olarak geliştirilmiştir.

## Kullanılan Teknolojiler

### Frontend

* React.js
* Material UI
* Axios
* React Router

### Backend

* Spring Boot
* Spring Security
* JPA / Hibernate
* REST API

### Veritabanı

* PostgreSQL

### Ödeme Sistemi

* Stripe (Demo Mod)

## Sistem Özellikleri

### Kullanıcı Modülü

* Ürün listeleme
* Ürün detay görüntüleme
* Sepete ekleme
* Sipariş oluşturma
* Sipariş geçmişi görüntüleme

### Yönetici Modülü

* Ürün ekleme
* Ürün güncelleme
* Ürün silme
* Stok yönetimi
* Sipariş yönetimi

## Kurulum

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

### PostgreSQL

```sql
CREATE DATABASE deutschbookdb;
```
